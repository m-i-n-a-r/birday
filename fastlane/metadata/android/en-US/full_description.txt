The app contains some basic personalization options and an introduction, plus some new resources like animated vector drawables and the automatic dark/light theme. Support and updates are guaranteed!

Well, I'll tell you 3 main reasons:
<ul>

<h2><b>Features ðŸŽ‚</b></h2>
<ul>
    <li> Simple events timeline, with the next date and the name of each person </li>
    <li> Normal/favorite/ignored system, to aggregate your favorite events and see more information </li>
    <li> Overview screen and a lot of basic and advanced settings </li>
    <li> Import birthdays from contacts or calendar with basic duplicate detection, select what to import</li>

    <li> Each favorite event has an handy note field </li>

    <li> Light, dark and black Amoled theme; Material3 Expressive </li>
    <li> Selectable accent and Monet accent (12 choices) </li>
    <li> Hide and unhide the top cards for maximum ui flexibility </li>
    <li> A quick search bar, yearly overview and type selector! </li>
    <li> 2 customizable widgets </li>
    <li> First introduction and beautiful animations including an animated notification icon </li>
    <li> Many possible customization and options </li>
</ul>

Please note: the notification system relies on the Android OS resources, thus some manufacturers may cause malfunctions. The reason is that some Xiaomi, Huawei and Oneplus devices kill the apps instead of simply closing it, or block the automatic startup of the app process. I can't fix this, but you can find more information on dontkillmyapp.com.
